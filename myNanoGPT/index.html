myNanoGPT Text Generator

Random Text Generated from Cleaned Data


    
  






  
This application is a locally hosted text-generation system trained on a custom real-world dataset.

  
  
Project highlights:

  

    • Cleaned and standardized the corrupt_movie_corpus.txt dataset, removing HTML artifacts, URLs, excessive punctuation, and other noise.

    • Adapted Andrej Karpathy’s nanoGPT framework to train a GPT model on the refined movie-corpus dataset.

    • Developed a Flask-based web interface for interacting with the trained model.
